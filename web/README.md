# Rifas MVP (frontend demo)

## Ejecutar local

```bash
cd web
python3 -m http.server 4173
```

Abrir: `http://localhost:4173`

## Incluye
- Selección manual de números con botones y búsqueda.
- Mínimo de compra (5) y sin máximo por transacción.
- Checkout con pago simulado.
- Descarga de comprobante PDF placeholder.
- Consulta de entradas por correo o documento.
- Panel admin demo para publicar número ganador principal y subpremio.
